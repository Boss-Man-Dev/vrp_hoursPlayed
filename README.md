# vrp_hoursPlayed
keeps track of total hours played. Updates every 15 minutes by default.

## Features:

### Command:
* default: /time
* can be changed in cfg
    
### Update:
* intrival: every 15 minutes
* can be changed in cfg
    
### Database:
* table Name: vrp_hours_played
* table created automatically
